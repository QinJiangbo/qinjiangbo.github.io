Guava区间Range

区间，有时也称为范围，是特定域中的凸性（非正式说法为连续的或不中断的）部分。在形式上，凸性表示对a<=b<=c, range.contains(a)且range.contains(c)意味着range.contains(b)。

Range的表示形式

区间 数学表达式
(a..b) {a < x < b}
[a..b] {a <= x <= b}
[a..b) {a <= x < b}
(a..b] {a < x <= b}
(a..+∞) {x > a}
[a..+∞) {x >= a}
(-∞..b) {x < b}
(-∞..b] {x <= b}
(-∞..+∞) 所有值

上面的a、b称为端点 。为了提高一致性，Guava中的Range要求上端点不能小于下端点。上下端点有可能是相等的，但要求区间是闭区间或半开半闭区间（至少有一个端点是包含在区间中的）。

区间构建

区间 方法
(a..b) open(C, C)
[a..b] closed(C, C)
[a..b) closedOpen(C, C)
(a..b] openClosed(C, C)
(a..+∞) greaterThan(C)
[a..+∞) atLeast(C)
(-∞..b) lessThan(C)
(-∞..b] atMost(C)
(-∞..+∞) all()
