总结

关于模型文件的保存是非常关键的一步，因为它关系到后面整体系统性能的好坏，试想一下，如果我们每一次都需要重新训练模型，那对于整个系统的性能伤害将是巨大的。所以，我比较建议先训练模型，保存为一个文件，然后通后面再过读取这个模型文件进行预测。这也是数据挖掘和机器学习的正规的方法论。
