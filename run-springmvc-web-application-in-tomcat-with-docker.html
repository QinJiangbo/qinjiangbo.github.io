  • Docker (https://store.docker.com/editions/community/docker-ce-desktop-mac)
  • Tomcat (http://www-eu.apache.org/dist/tomcat/tomcat-8/v8.5.23/bin/apache-tomcat-8.5.23.tar.gz)
